WhatsApp Automation System

Descripción;
Este proyecto corresponde a un sistema de automatización de envíos de mensajes por WhatsApp, desarrollado a medida según los requerimientos del cliente.
La solución permite realizar envíos automáticos bajo reglas predefinidas, utilizando un servicio externo de mensajería.


Funcionalidad principal;
Envío automático de mensajes de WhatsApp
Configuración personalizada según el caso de uso solicitado
Ejecución continua mientras el sistema se encuentre activo
La operación del sistema se realiza a través de una página web, que constituye la interfaz principal para el uso del producto


Requisitos para su funcionamiento;
Cuenta activa en un servicio externo de mensajería (Twilio)
Número de WhatsApp habilitado por dicho servicio
Servidor con conexión permanente a internet
El uso del servicio externo puede implicar costos adicionales, que dependen exclusivamente del proveedor.


Alcance de la entrega;
Este proyecto se entrega como una solución puntual y cerrada, desarrollada para cubrir un requerimiento específico.


La entrega incluye;
Código fuente del sistema
Configuración base
Página web de operación


No incluye;
Soporte técnico continuo
Mantenimiento evolutivo
Adaptaciones futuras
Garantía frente a cambios o bloqueos del proveedor externo
Cualquier modificación o ampliación deberá acordarse por separado.


Limitaciones;
El sistema depende de servicios externos para el envío de mensajes
No se trata de una integración oficial directa con WhatsApp
El uso indebido puede generar restricciones o bloqueos por parte del proveedor


Uso del repositorio;
Este repositorio se entrega como respaldo del desarrollo y referencia del proyecto.
No está orientado a su modificación por parte del usuario final.
